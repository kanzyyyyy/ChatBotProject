#Files

main.py : run this to start the chatbot
oracle.py : detects emotions from user input
chatbot.py : picks the right response based on emotion and strategy
strategies.json : defines the strategies used by chatbot.py

#Commands

#Start the chatbot loop : python3 main.py

#Test emotion detection only : python3 oracle.py

#Test the chatbot only : python3 chatbot.py

Type quit to exit at any time.
